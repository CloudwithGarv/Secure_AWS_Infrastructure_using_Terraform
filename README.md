# Secure AWS Infrastructure using Terraform

Designed and deployed a production-ready AWS environment using Infrastructure as Code (IaC) principles. The project provisions networking, compute, database, monitoring, and security components through reusable Terraform modules.

## Features

- VPC with Public and Private Subnets
- EC2 Application Servers
- Amazon RDS Database
- Application Load Balancer
- IAM Roles and Policies
- CloudWatch Monitoring
- Security Groups and Network Segmentation
- Terraform Modular Architecture
